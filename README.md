# Point Text Helper
"Point Text" Helper for ThreeJS

## scripts
```shell
npm run generate-atlas
```

compile:
```
tsc
```

## Run the tests/examples
3 process required for the time being:
- typescript
```
tsc --watch
```
- webpack
```
cd tests/examples
webpack --watch
```
- serving
```
npx serve
```
