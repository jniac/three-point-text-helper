# Point Text Helper
"Point Text" Helper for ThreeJS

## scripts
```shell
npm run generate-atlas
```

compile:
```
tsc
```

## Run the tests/examples
2 process required for the time being:
- run the library itself + static server
```shell
npm run dev
# (equivalent to)
ts-node extras/dev.ts
```
- webpack
```
cd tests/examples
webpack --watch
```
